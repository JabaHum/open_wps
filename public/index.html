WPS Logger

WiFi signal strength logger framework for WiFi Positioning System (WPS)

About your application’s environment

Getting started

Here’s how to get logging:

  1. supported environment

    We are currently supporting only Firefox as a client browser.

  2. download extension

    You can get WiFi and GPS data by using Firefox extension wps_logger_extension. If you need GPS data, you are also required to install gpsd. gpsd does not support Windows OS. gpsd-4-win is packaged based on version 2.37 and unfortunately it does not support JSON base commands. Currently only NMEA 0183 protocol is supported to get GPS data using serial connection.

  3. Now you are ready to record WiFi signal log!

    Goto map page to register your own building map, then collect WiFi logs for each map.
